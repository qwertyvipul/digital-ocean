# digital-ocean
Tips &amp; tricks for getting along DigitalOcean

### Getting started
```bash
sudo apt-get update
sudo apt-get upgrade
```

### Creating new user
```bash
# Add new user
adduser /home/testuser testuser

# Set password for new user
passwd testuser
```

Login to new user using putty

### Installing node
```bash
su root
apt install node
apt install npm
```
